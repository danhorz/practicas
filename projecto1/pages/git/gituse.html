Git Comandos

Los siguientes comandos son una instruccion basica de uso de Git:

CONFIGURACION GLOBAL GIT

git config --global user.name 'TU NOMBRE DE USUARIO GIT'
git config --global user.email 'TU CORREO ELECTRÓNICO'
git config --global core.editor "code --wait"
git config --global init.defaultbranch main"

Las anteriores lineas de código representan la configuracion general de git como el nombre de usuario nuestro correo y es el medio de enlace entre nuestraa PC a nuestro repositorio. Sin embargo, para que este enlazado debemos auntenticar mediante los si guientes comandos y acciones:

CLONAR UN REPOSITORIO GIT

git clone "enlace de tu repositorio"

Selecciona una carpeta en tu ordenador para importar los archivos de tu repositorio Git, luego haz click derecho y selecciona la terminal Git y colocar el codigo anterior mas tu enlace.

Inializar GIT en una carpeta manualmente 

git init

Eliminar el archivo Git de tu proyecto en el ordenador 

rm -rf .git

Subir contenido a Git y Publicarlo

git add .
git commit -m "comentario"
git push

El comando git add es para logar subir los archivos al repositorio. El comando commit es un punto de guardado previo a la publicacion para el repositorio. Y git push muestra los cambios en el repositorio.
