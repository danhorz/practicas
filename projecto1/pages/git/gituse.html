INSTALAR GIT-BASH(TERMINAL)

Descarga GitBash Entra la página de descargas de Git. Haz click en el botón download. Una vez que se descargue el instalador, abre el archivo y comienza con el proceso de instalación. Instala GitBash Una vez que comience la instalación, se te aparecerán una serie de ventanas que te pedirán que elijas algunas cosas dentro de la configuración, luego presiona el botón “Next” para continuar con el proceso. Utiliza estas notas como guía: Select Components: Selecciona todas las opciones. Choose Default Editor: Selecciona Notepad. Initial Branch: Escoge la opción “Override” y deja main por defecto. Path: Escoge “Git from the command line and also from 3rd party software”. OpenSSH: Elige “Use bundled OpenSSH”. OpenSSL: Elige “Use the OpenSSL Library”. Line Ending Conversions: Elige “Checkout Windows-style, commit Unix-style”. Terminal Emulator: Elige “Use Windows Terminal”. Default Behavior for ‘git pull’: Elige “Fast Forward or Merge”. Credentials Manager: Elige “git credential manager”. Extra Options: Elige solo “Enable filesystem caching”. Experimental Options: No marques ninguna de las opciones. Cuando se termine la instalación, marca la opción “Launch GitBash” y haz click en el botón “Finish”.

CONFIGURACION GLOBAL GIT

git add .
git commit -m "feat: initial commit"
git push origin main
