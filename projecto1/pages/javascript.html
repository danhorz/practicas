Variables en JavaScript

Variables


                //VARIABLE LET
                let variableCambiable=0;
                    variableCambiable=1;
                //VARIABLE CONST    
                const variableNoCambiable=0;

VARIABLE LET: Uso exclusivo en un bloque de código capaz de cambiar su valor a futuro.
VARIABLE CONST: Uso exclusivo en un bloque de código sin poder cambiar su valor.
¡WARING!
Existe una variable llamada var , pero NO SE ACONSEJA USARLO por el hecho de que se puede instanciar el mismo prefijo var con el mismo nombre.

Acceder a información general del documento

Cambiar título del documento

 console.log(document.title); 
 document.title = "Nuevo Título";

Obtener URL de la página:

console.log(document.URL);

Obtener dominio 

 console.log(document.domain);

Obtener formularios:

 let forms = document.forms; 
 console.log(forms);  // Muestra una colección de todos los formularios

Acceder a un formulario específico y sus elementos

 let form = document.forms["nombreFormulario"];  // O document.forms[0] para el primer formulario
 let input = form.elements["nombreCampo"];  // Accede a un campo específico por su nombre

Obtener todas las imágenes en la página

 console.log(images);

Obtener todos los enlaces en la página:

let links = document.links;
 console.log(links);

Crear y agregar elementos al DOM

Insertar un nuevo elemento dentro de otro

let nuevoElemento = document.createElement("p");
nuevoElemento.innerText = "Soy un nuevo párrafo";
document.body.appendChild(nuevoElemento);  // Lo agrega al final del `body`

Insertar un elemento antes de otro

let nuevoElemento = document.createElement("p");
 let referencia = document.getElementById("elementoReferencia");
document.body.insertBefore(nuevoElemento, referencia);

Trabajar con estilos CSS de los elementos

Cambiar estilos múltiples

let elemento = document.getElementById("miElemento");
 elemento.style.cssText = "color: red; font-size: 20px;";

Creación de fragmentos de documentos (para mejorar el rendimiento)

Usar fragmentos para evitar repintados múltiples

let fragment = document.createDocumentFragment();

                
                    for (let i = 0; i < 10; i++) {
                        let nuevoElemento = document.createElement("p");
                        nuevoElemento.innerText = `Elemento ${i + 1}`;
                        fragment.appendChild(nuevoElemento);
                    }
                    
                
document.body.appendChild(fragment);
